# Sound files placeholder

This directory should contain the following sound files:

- `pickup.mp3` - Card pickup sound
- `drop.mp3` - Card drop success sound
- `invalid.mp3` - Invalid action sound
- `flip.mp3` - Card flip sound
- `safe.mp3` - SAFE reveal sound (chime + sparkle)
- `doom.mp3` - DOOM reveal sound (crack + bass hit)
- `elim.mp3` - Elimination sound (whoosh down)
- `win.mp3` - Victory fanfare
- `turn.mp3` - Turn notification
- `swap.mp3` - Swap action sound

## Recommended sources for free sound effects:

- [Freesound](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Zapsplat](https://www.zapsplat.com/)

Keep sounds short (under 1 second for most) and lightweight for mobile performance.
